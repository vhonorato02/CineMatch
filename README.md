# CineMatch - Film Noir Edition

Tinder de filmes para casais, com estética Film Noir minimalista (preto e branco).

## Deploy Rápido no Vercel

1. Acesse [vercel.com](https://vercel.com)
2. Faça login com GitHub
3. Clique em **"Add New Project"**
4. Selecione o repositório **CineMatch**
5. Clique em **Deploy**

Pronto! Em 2 minutos você tem a URL do app.

## Features (em desenvolvimento)

- ✅ Interface Film Noir (preto/branco/cinza)
- ⏳ P2P pairing via PeerJS
- ⏳ Swipe de filmes
- ⏳ Matches sincronizados
- ⏳ Firebase sync opcional

## Tecnologias

- React 18
- Vite 5
- TypeScript
- Firebase
- PeerJS
